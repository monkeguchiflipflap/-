# Championship

Материалы от организаторов чемпионата.

- `case.md` — текст кейса (условие задачи)
- `criteria.md` — критерии оценки решения
- `context.md` — дополнительный контекст: регламент, дедлайны, формат защиты и т.д.
- `past_winners/` — работы победителей прошлых лет (опционально, для референса)
