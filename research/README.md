# Research

Факты и тезисы, собранные в ходе ресёрча. Каждый факт — со ссылкой на источник.

Рекомендуемый формат файла: один markdown-файл на тему/направление ресёрча (например, `market_overview.md`, `competitors.md`, `target_audience.md`), внутри — список тезисов вида:

```
- Тезис/факт. Источник: [название](ссылка)
```
