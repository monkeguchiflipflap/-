# Alfa Case Championship — рабочий репозиторий

Репозиторий команды для решения кейса чемпионата Альфа-Банка. Используется как единый источник правды: версионирование материалов, совместная работа, история изменений.

## Структура

- `championship/` — условия кейса, критерии оценки, любой контекст от организаторов; `championship/past_winners/` — работы победителей прошлых лет (опционально, для референса)
- `research/` — ресёрч: факты, тезисы, цифры — обязательно со ссылками на источники
- `model/` — модель/расчёты, лежащие в основе решения (данные, выкладки, результаты)
- `meetings/` — транскрипты и заметки командных встреч
- `presentation/` — итоговое решение: `presentation/slides/` — один markdown-файл на слайд (для точечной доработки и оценки), плюс общий сборный файл всей презентации в корне папки

## Принцип работы

Презентация не генерируется целиком одним промптом — тезисы и структура собираются в markdown, а сама презентация (визуал) верстается отдельно на основе этих файлов.
